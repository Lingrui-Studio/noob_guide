# 招新安排

## 面向群体

电子科技大学在读大一大二学生（不限学院专业）

## 招新目标

凌睿工作室对招新的选拔标准是：

- 热爱编程
- 有较强学习能力
- 勇于向困难问题发起挑战
- 与凌睿成员合得来

以上几点，招新参与者会在招新期间展现出来。通常来说，整个招新过程中参与者会越来越少，不是因为我们在中途淘汰了一些同学，而是因为很多人中途自己放弃了。拿去年举例，一轮面试名单公示之后，有三位同学觉得竞争太大主动放弃了面试资格，然而我们一面基本没有淘汰人。

我们不会设置具体的招新人数，但是会在招新时给出较明确的进面标准，只要你踏踏实实学习并完成任务，坚持到最后，基本都能加入我们。若是某一方向的同学都做得很差，那这个方向可能只招一两个甚至不招，若是都做得很好，那招八九个也是有过的。

> 不要觉得自己排名靠前就稳了，每年都有排名第一的同学最后面试时被淘汰，自己几斤几两自己心里有数。

## 编程小白有机会加入工作室吗

诚然，有部分同学在入学前就有了编程经验，或是参与过信息学竞赛（可能甚至拿过奖）、或是玩过 Scratch 这类图形化编程、或是觉得 QQ 机器人很酷自己写过脚本、或是为自己热爱的游戏写过 mod、又或是选考信息技术的浙江同学……这样的同学若是真心热爱编程，那当然很好！然而，有经验固然可以短暂地领先，却不一定能一直保持领先；而且这样的同学毕竟是少数，历史经验告诉我们，每年入选的成员中仍然是最开始“零基础”的同学占大半。所以与其担心别人快人一步，不如静下心来好好探索丰富多彩的程序世界~
